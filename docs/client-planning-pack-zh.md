# 爱尔兰华人志网站规划说明

## 1. 项目目的

本文档用于配合第一阶段网站演示版本，向客户说明网站的整体方向、内容结构和首页规划。

本阶段的目标不是取代现有微信公众号，而是为公众号内容提供一个更清晰、更长期、更便于展示和合作的网站入口。网站可以承载新闻、专题、视频、活动、留学指南和商务合作信息，让现有内容从微信生态延伸到更完整的媒体平台。

## 2. 项目定位

爱尔兰华人志可以定位为面向在爱华人的中文社区媒体平台，服务对象包括在爱居民、留学生、家庭、专业人士、游客和本地商户。

核心价值包括：

- 提供可靠的爱尔兰中文资讯
- 提升爱尔兰华人社区的可见度
- 沉淀生活、留学、旅游、活动等实用内容
- 为本地商户、社团和合作伙伴提供专业展示入口

商业化可以作为媒体平台自然延伸的一部分，但不建议把网站一开始就包装成纯广告平台。更好的方向是先建立可信内容，再逐步发展推广、广告和服务合作。

## 3. 目标用户

- 在爱尔兰生活的华人居民
- 准备来爱尔兰或已经在爱尔兰学习的中国学生
- 需要本地生活资讯和周末活动建议的华人家庭
- 刚到爱尔兰、需要租房、签证、银行卡、交通等实用指南的新居民
- 希望触达华人客户的本地商户
- 社区组织、活动主办方和文化合作伙伴

## 6. 后续功能与内容方向

### 本地商户推广专区

网站后续可以增加本地商户推广专区，帮助餐饮、留学服务、旅游服务、房产、法律、会计、生活服务等商户触达在爱华人用户。

可考虑的合作形式：

- 商户软文介绍
- 商户展示页
- 首页推荐位
- 活动赞助
- 节日营销专题
- 微信公众号与网站联合推广套餐

### 本地旅游与向导匹配

平台可以进一步发展本地旅游建议和向导匹配功能，服务来爱游客、留学生、探亲家庭和商务访问人群。

可匹配的需求包括：

- 亲子周末路线
- 学生低预算路线
- 中文本地向导
- 接机与落地第一周协助
- 探亲家庭定制路线
- 文化、美食和城市体验路线

这个方向既可以形成服务收入，也可以持续产生旅游和生活类内容。

### 短视频信息流

网站后续可以加入类似 TikTok 的短视频信息流，用更轻量的方式吸引用户注意力，也为广告和商户展示提供更多空间。

建议包装方向：

- 社区短新闻
- 活动现场花絮
- 一分钟生活指南
- 留学生日常
- 商户短视频展示
- 快速人物采访

短视频内容最好与微信公众号、TikTok、Instagram、小红书等平台联动，不建议一开始就建立过重的独立视频生产流程。

## 7. 第一阶段范围

当前阶段建议保持简洁，重点展示：

- 网站视觉方向
- 核心栏目结构
- 首页内容布局
- 微信公众号内容如何延伸为网站文章、视频、指南和活动页
- 后续广告与商务合作的基础空间

## 8. 建议下一步

1. 确认最终栏目和首页板块。
2. 用真实微信公众号内容替换演示文章。
3. 选择首批 10-15 篇文章或公众号内容进行迁移。
4. 制定本地商户合作套餐。
5. 决定下一阶段优先做内容管理、广告位，还是短视频信息流。
